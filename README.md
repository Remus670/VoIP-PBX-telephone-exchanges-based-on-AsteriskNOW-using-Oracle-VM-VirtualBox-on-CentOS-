This project implements a complete VoIP telephony system using Asterisk, configuring both SIP and IAX protocols for communication. It defines multiple call routing rules, contexts, and extensions to manage incoming, internal, and restricted calls. The system includes advanced features such as call queues, conference rooms (MeetMe), and an interactive voice menu with various audio functions. It also integrates security mechanisms like PIN authentication and restricted number access handling. Overall, the project demonstrates how to design and manage a flexible and functional IP telephony network.
